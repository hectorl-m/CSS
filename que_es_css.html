Introducción a CSS

1. ¿Qué es CSS y para qué sirve?

CSS (Cascading Style Sheets) es un lenguaje que se utiliza para definir el estilo y diseño visual de las páginas web. Separa el contenido HTML de su apariencia, permitiendo una fácil personalización y mantenimiento del diseño de una web.

2. Formas de aplicar CSS en HTML

CSS en línea (inline)

Se aplica directamente en el atributo style de un elemento HTML:

<h1 style="color: blue;">Título con CSS en línea</h1>

CSS en el encabezado (internal)

Se coloca dentro de la etiqueta <style> en la sección <head> del documento HTML:

<style>
  h1 { color: green; }
</style>

CSS externo

Se vincula un archivo CSS externo con la etiqueta <link>:

<link rel="stylesheet" href="estilos.css">
