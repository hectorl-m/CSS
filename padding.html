Padding, Margin, Border, Width and Height

Padding (Relleno)

El padding es el espacio entre el contenido de un elemento y su borde. En este caso, el padding es de 20px, lo que significa que el contenido está separado del borde por ese espacio.

Margin (Margen)

El margin es el espacio fuera del borde de un elemento. En este caso, el margen es de 30px, lo que separa este elemento de otros elementos circundantes.

Border (Borde)

El border es el borde del elemento, y en este caso es de 5px de ancho, de color azul.

Width (Ancho) y Height (Altura)

El width es el ancho de un elemento, y el height es la altura. Aquí, el ancho es de 200px y la altura es de 150px. Estos valores definen el tamaño del área de contenido del elemento.
