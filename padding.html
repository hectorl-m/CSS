Padding, Margin, Border, Width and Height

Padding (Relleno)

El padding es el espacio entre el contenido de un elemento y su borde.

Margin (Margen)

El margin es el espacio fuera del borde de un elemento.

Border (Borde)

El border es el borde del elemento, y en este caso es de 5px de ancho, de color azul.

Width (Ancho) y Height (Altura)

El width es el ancho de un elemento, y el height es la altura.
